Afinal o que é o Anime Store?

Nosso projeto foi desenvolvido como parte do curso Analise e Desenvolmento de Sistema, relacionado a materia de Cliente-Servidor da PUC Goiás, sob a orientação do professor Fernando Abadia. O principal objetivo deste projeto é capacitar os alunos a desenvolver conexões com bancos de dados e criar a interface front-end em Java, porém nós optamos pelo Desenvolmento Web (JavaScript, CSS e HTML) na parte front-end e Java Back-end. No mundo atual, os animes conquistou uma enorme base de fãs, proporcionando histórias emocionantes, como personagens carismáticos e uma infinidade de produtos colecionáveis. Com isso em mente, decidimos criar um site de vendas fictício, totalmente dedicado ao universo dos animes. Nossa equipe se esforçou para criar uma experiência de compra virtual, onde os usuários podem explorar uma vasta gama de produtos relacionados a animes. Oferecemos, mangás impressos, action figures, acessórios de cosplay e muito mais. Além disso, nossa equipe dedicou-se a criar uma interface intuitiva e agradável para tornar a sua experiência de compra ainda mais prazerosa. Gostaríamos de agradecer a todos que tornaram este projeto possível e, em especial, ao professor Fernando Abadia por sua orientação e apoio ao longo do desenvolvimento do site. Esperamos que você aproveite sua visita ao nosso site de venda fictício de anime e encontre os produtos que deseja para completar sua coleção. Sinta-se à vontade para entrar em contato na aba contato conosco caso tenha alguma dúvida ou comentário. Divirta-se explorando o maravilhoso mundo dos animes em nosso site!
